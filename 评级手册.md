# 苏州中学树莓社成员能力评级指南

## 一、评级等级说明（通用）

| 等级     | 含义           | 树莓社应用提示                                                                                                                                                                                                                  |
| :----- | :----------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **∞**  | 超越常规评价体系，不可比 | 极少出现，通常用于创社元老或不可复制的基石性贡献（如定义社团文化、创建核心制度、获全球顶级专业认证）∞等级的使用遵循以下原则：每个成员在同一届评级周期内，最多只能在一个维度上获得∞。∞必须同时满足两个条件：① 该维度表现远超同龄人常规水平；② 该贡献在社团历史上具有不可替代的唯一性（如创社、定义文化内核、获得全球顶级专业认证等）。若某成员有多个维度达到∞水准，则选择其中最具代表性的一项授予∞，其余维度按A+评定，并在备注中说明。 |
| **A+** | 接近顶级水平，接近卓越 | 略低于∞但已接近最高水平；用于当某成员有多个维度都达到∞水准时，除最具代表性的一项授予∞外，其余维度按A+评定；或用于该维度表现极其接近∞但尚未达到"不可替代"的严格标准 |
| **A**  | 该维度达到顶级水平    | 核心优势，可称为"招牌能力"；常见于历届组委会核心成员                                                                                                                                                                                              |
| **B**  | 明显高于平均水平     | 稳定优秀，成熟骨干的常见等级                                                                                                                                                                                                           |
| **C**  | 标准、均衡水平      | 大多数合格社员的基准线（能够完成常规任务，无硬伤）                                                                                                                                                                                                |
| **D**  | 存在明显短板       | 需要刻意补强或依赖团队其他维度弥补                                                                                                                                                                                                        |
| **E**  | 极低水平         | 接近没有实际作用                                                                                                                                                                                                                 |
| **？**  | 无法测定 / 尚未定型  | 新社员、转型期角色或信息不足                                                                                                                                                                                                           |
| **无**  | 完全不存在该能力     | 从未建立或无人负责                                                                                                                                                                                                                |
| **完成** | 已发展成熟、无成长空间  | 一般用于社团整体维度，不用于个人                                                                                                                                                                                                         |

> **关于 A 与 ∞ 的界定（与章程发展路线挂钩）**\
> 章程序言明确规定的发展路线，是对社团领袖的核心要求。该路线标准本身极高。
>
> - **A 级**：个人在该维度上**完全达到**发展路线所提出的要求。例如：建立民主集中制的组织体系（组织）、产出维护影视核心地位的微电影（创作）、营造百花齐放的文化氛围（文化）等。
> - **∞ 级**：个人在该维度上**显著超出**发展路线的要求，表现为**创造或定义了发展路线本身**（如制度体系设计）、**获得外部顶级权威认可**（如世界顶尖专业院校录取）、或做出了**不可替代的范式奠基**。∞ 级必须同时满足"远超同龄人"和"贡献不可替代"。\
>   注意：达到发展路线要求即为 A，因此 A 已是卓越；∞ 仅用于极少数超出预期的情形。

## 二、六个维度定义与评级锚点

### 1. 组织

**内涵**：团队建设、制度执行、活动策划与落地、分工协调、项目管理。\
**文件依据**：

- 历史决议第五部分"社团管理体制改革"详细描述了从"家长制"到"民主集中制"的转变。
- 章程第五条："社团管理实行民主集中制的原则……组织委员会由民主选举产生。"
- 发展史中"树莓奇迹夜"（2024奥体运动会前夜通宵改方案）为应急组织能力的典范。

| 等级    | 典型行为描述（树莓社语境）                                                                         |
| :---- | :------------------------------------------------------------------------------------ |
| **∞** | 从无到有创建社团，奠定组织基础                                                                       |
| **A** | 建立并完善民主集中制、组委会制、招新换届等制度体系，完全达到发展路线要求；独立策划并执行全校级大型活动；担任组委会核心成员，带领跨部门团队完成复杂任务；设计出高效率新流程 |
| **B** | 独立负责一个子活动（如一次摄影展、一次社团例会）；担任部门部长，能够调动本部门成员完成任务；熟练使用项目管理工具（如飞书、共享表格）                    |
| **C** | 参与社团活动并承担明确分工（如做一次活动的摄影组长）；能够按流程完成分配的工作；了解社团组织架构                                      |
| **D** | 经常缺席组织会议；无法完成分配的协作任务；分不清部门职责                                                          |
| **E** | 从未参与任何组织事务                                                                            |
| **？** | 新社员尚未参与组织工作                                                                           |
| **无** | 从未成立任何组织（不适用于个人）                                                                      |

***

### 2. 创作

**内涵**：数字媒体作品的产出质量、艺术表达、技术完成度。包括微电影、纪录片、摄影作品、短视频、广播节目、后期特效等。\
**文件依据**：

- 章程序言："微电影（影视创作）业务是苏州中学树莓社的根本业务。"
- 历史决议第二部分列举了《苏迷》纪录片、FilMarathon2019、传媒中心节目等成果。
- 发展史中《识茶记》（2022）、《数据之下》（2022）、《寻找》（2025）等微电影，以及B站播放量破8万的毕业典礼视频。

| 等级    | 典型行为描述                                                                                               |
| :---- | :--------------------------------------------------------------------------------------------------- |
| **∞** | 作品被全球顶级专业院校作为录取依据（如马雨璋入读NYU帝势艺术学院影视专业）；作品获得国家级以上显著认可                                                 |
| **A** | 主创作品获得市级以上奖项或在权威平台（如苏州日报、B站首页）获得高曝光；独立完成一部完整微电影并获广泛好评；摄影作品被官方选为校庆、毕业典礼主视觉；在社团年度创作中起核心作用（如《识茶记》制片或导演） |
| **B** | 主创作品在社团B站/公众号获得较好数据（播放量过万、点赞过千）；能够独立完成从策划到后期的完整短片；作品被学校公众号采用多于5次                                     |
| **C** | 能够完成技术达标的作品（曝光正确、剪辑流畅、声音清晰）；参与过至少一次微电影剧组（任何岗位）；提交过不少于3次活动照片且被选用                                      |
| **D** | 作品常有技术硬伤（虚焦、过曝、声画不同步）；无法独立完成简单拍摄/剪辑任务                                                                |
| **E** | 从未产出任何作品                                                                                             |
| **？** | 刚学习拍摄/剪辑，尚未产出完整作品                                                                                    |
| **无** | 无作品记录                                                                                                |

***

### 3. 文化

**内涵**：理解、维护与传递树莓社核心价值观、历史故事、社团氛围。包括熟悉社史、参与文化传承活动（老带新、社员大会）、弘扬"用镜头留住温度，用影像传承记忆"等口号、参与树莓酱IP或文创设计。\
**文件依据**：

- 历史决议第一部分（八）："树莓社积极建设并发扬社团文化……以'树莓'为核心视觉元素。"
- 章程序言中详细记录了社团文化的起源（张诗菡创造"树莓娘"形象，沈孙丰提出用"树莓"代替数字媒体）。
- 发展史中2024年国旗下演讲《用流动的影像传承历史》系统总结六年文化。
- 附录E将"树莓文化和社团宣传建设"贡献者单独列出。

| 等级    | 典型行为描述                                                                                            |
| :---- | :------------------------------------------------------------------------------------------------ |
| **∞** | 提出并定义社团核心文化符号（该符号被长期沿用且写入章程）                                                                      |
| **A** | 系统整理社团历史，撰写发展史并作为章程附件；在重大场合（如国旗下演讲）系统输出社团文化，影响全校；经常向新社员讲述社团历史与文化；组织文化类活动；设计并推广树莓酱IP或重要文创；担任核心传承角色 |
| **B** | 积极参与社员大会、换届仪式；主动带新社员熟悉社团规则；在作品或活动中主动体现树莓文化元素                                                      |
| **C** | 了解社团基本历史；**能够向校外同学或新社员准确介绍树莓社的基本情况**；能说出树莓社口号和主要成就；遵守并维护社团原则。                                     |
| **D** | 对社团文化漠不关心；多次违反社团原则（如私自外泄未发布作品、**违反数据安全规定**）。                                                      |
| **E** | 完全不了解社团文化                                                                                         |
| **？** | 新社员尚未接触文化传承                                                                                       |
| **无** | 完全不关心文化                                                                                           |

***

### 4. 影响

**内涵**：社团或个人的对外传播力、知名度、资源链接能力。包括运营新媒体（公众号/B站/小红书/抖音）、接受媒体报道、促成跨校合作（树莓派联盟）、拉取赞助或吸引大量新社员。\
**说明**：影响维度以**集体影响为主（70%）**，个人相关影响为辅（30%）。集体影响包括社团获奖、媒体报道、跨校联盟、新媒体账号运营数据（粉丝、阅读量）、招新规模等。个人影响需以树莓社成员身份获得（如以社团名义发表作品、接受采访、获得专业录取等），纯个人账号或成就不计入本维度。\
**文件依据**：

- 历史决议第六部分（三）提到沈孙丰任苏州市社团联盟主席、创建树莓派苏外分部等。
- 章程序言："树莓派·苏州数字媒体学生社团联盟签约成立大会……12所高中社团达成合作协定。"
- 发展史中公众号粉丝破千、B站播放量破10万、国旗下演讲推文阅读量过万等。

| 等级    | 典型行为描述                                                                                                                      |
| :---- | :-------------------------------------------------------------------------------------------------------------------------- |
| **∞** | 创建跨校联盟，影响超过5所外部学校，并促成长期合作机制；或主导社团获得市级以上官方荣誉且贡献具有不可替代性                                                                       |
| **A** | 主导一次市级以上媒体报道；运营的官方账号粉丝增长超1000，单篇阅读量破万，**或达到该年度全校学生社团传媒数据的天花板水平**；演讲或展示触达超1万人次（如周熠嘉国旗下演讲推文破万）；拉取较高金额赞助或物资；凭借个人魅力招新吸引超过30名新社员 |
| **B** | 运营账号单篇推送阅读量超3000；促成一次与校外组织或社团的正式合作；招新吸引超过30名新社员                                                                             |
| **C** | 参与运营社团账号，定期发布内容；贡献过被学校公众号选用的素材；成功邀请朋友加入树莓社                                                                                  |
| **D** | 从不转发社团内容；没有外部联系能力                                                                                                           |
| **E** | 对外无人知晓                                                                                                                      |
| **？** | 新账号或新渠道探索期                                                                                                                  |

***

### 5. 责任

**内涵**：奉献精神、志愿服务、危机担当、遵守承诺。包括参与摄影志愿服务、主动承担额外工作、维护社团利益、在困难时刻挺身而出。\
**文件依据**：

- 历史决议第三部分记录了与社管中心的冲突及沈孙丰"力挽狂澜"。
- 发展史中2020年抗疫募捐（"募集数十万元善款"）、2024年"树莓奇迹夜"通宵改方案。
- 章程序言："树莓社摄影志愿服务"成为品牌，石允涵创立此模式。
- 树莓社口号："面对挑战，我能行；遇到困难，我不怕；突发状况，我担当。"

| 等级    | 典型行为描述                                                             |
| :---- | :----------------------------------------------------------------- |
| **∞** | 在重大危机中挺身而出并改变局面（如2020抗疫募捐核心组织者）                                    |
| **A** | 每学期志愿服务时长全校前列；主动承担未被分配的任务（如整理素材库、维修设备）；在活动突发状况中解决问题（如"树莓奇迹夜"通宵改方案） |
| **B** | 按时完成本职任务；愿意在紧急情况下加班（如活动后熬夜剪辑）；不推卸责任                                |
| **C** | 完成分配的任务，不拖延；不造成额外麻烦；**严守公私边界，不因个人情绪或私人关系影响工作**。                    |
| **D** | 经常迟到、缺席或无故爽约；任务质量低；**在管理中夹带私私怨，影响社团公允**。                           |
| **E** | 从不承担责任；**引发重大舆论或安全事故且逃避追责**。                                       |
| **？** | 尚未有责任考验                                                            |
| **无** | 完全不负责任                                                             |

***

### 6. 创新

**内涵**：引入新技术、新工作流、新活动形式、新传播方式。不等同于"技术"，而是"创造性地解决问题"。

1. **工作流创新**：优化协作机制，打破部门壁垒，提升运转效率。
2. **形式创新**：策划全新的活动模式、拍摄题材或社团互动方式。
3. **传播与文化创新**：拓宽新媒体渠道、开创全新的受众交互方式、将抽象文化实体化（如IP开发）。
4. **技术创新**：引入新设备、新软件以攻克特定视听表达难题。

**文件依据**：

- 历史决议第五部分提到照片直播、视频直播等创新。
- 发展史中：引入“照片直播”（石允涵，工作流与传播创新）、视频直播标准化（顾衡庭）、“共享相册”与“画纸采访”（王嫣然，活动形式与交互创新）、“树莓酱”IP体系与小红书运营（朱璟煊等，品牌与渠道创新）。
- 章程序言："照片直播业务是树莓社新闻传播业务的重要发展成果。""第七届组委会推动的“治理体系数字化转型”与“创意孵化体系”（杨梓言、郑楚琨，底层机制创新）。"

| 等级    | 典型行为描述                                                                                                                                      |
| :---- | :------------------------------------------------------------------------------------------------------------------------------------------ |
| **∞** | 提出并落地一种被全社团沿用数年的创新模式（如照片直播、共享相册、树莓酱IP体系或视频直播标准化流程）                                                                                          |
| **A** | \*\*\[重大突破] 在核心领域实现“从无到有”的跨越，或用创造性思维解决棘手难题。\*\*在活动中首次引入新技术（如多机位直播、AI辅助剪辑）；设计全新的活动形式（如"画纸共绘""创意短片挑战"）；开发社团新媒体渠道并实现明显增长                       |
| **B** | \*\*\[优化改良] 在现有基础上进行行之有效的“微创新”。\*\*改进现有流程（如优化照片直播审核效率）；提出新颖的拍摄创意并被采纳；尝试新软件或设备并成功应用;面对现场突发设备或环境限制，能想出巧妙的“平替”或绕行方案（创造性解决问题）；主动学习并成功应用新工具解决痛点。 |
| **C** | \*\*\[管理层基本要求] 具备开放心态与基本的破局意识。\*\*愿意学习新工具；偶尔提出小改进建议，愿意主动适应；不再仅凭直觉做事，开始思考“有没有更好的方法”。                                                         |
| **D** | 墨守成规，排斥变化；在创作与执行中只愿做“机械执行者”，缺乏主观能动性。                                                                                                        |
| **E** | 不接触任何新技术；甚至以“以前都是这么干的”为由，阻碍团队采用更优的解决方案。                                                                                                     |
| **？** | 尚未遇到需要独立解决新问题的场景。                                                                                                                           |
| **无** | 毫无创新意识                                                                                                                                      |

## 三、使用方式

1. **个人雷达图评价**：根据被评者近期（一学期或一学年）的实际表现，对照上述锚点给出等级。
2. **整体评级均衡性**：优秀社员通常在3个以上维度获得B及以上；传奇领袖一般有多项A或∞。
3. **成长跟踪**：可每学期评级一次，观察维度提升情况。
4. **团队组建参考**：根据成员雷达图，搭配互补角色（如组织型配创作型，责任型配创新型）。

## 五、对社长/组委会成员的能力期望参考

章程并未要求社长在所有维度达到 A。根据章程第43条及发展路线，合格社长的基准线为：

- **影响、责任 不低于 B**
- **其余维度 不低于 C**

若有多项达到 B 或更高，可视为优秀。有 ∞ 或全 A 为传奇人物。

此标准也可用于组委会成员的自评与选拔参考。

**【阅表指南：如何理解树莓社的领导力模型？】**
在树莓社的名人堂雷达图中，分数的起伏不仅代表个人能力，更折射了不同历史时期对社团领袖的时代要求：

1. **全A型（六边形战士）**：如石允涵。代表了苏州市级顶尖社长的天花板水平，能够在各项业务中完美平衡，是社团冲击外部荣誉的究极形态。
2. **全B型（全能中流砥柱）**：这是树莓社优秀社长、校级十佳社长的标准画像。没有明显短板，能保障社团稳定繁荣。
3. **偏科型领袖（伟大的破局者）**：部分社长在“创作”等硬性业务维度仅为 C，但这绝非平庸。例如第五任社长李谦、第六任社长周熠嘉。在社团面临转型的关键期，他们主动放弃了个人的专业艺术创作（创作 C），将全部心血倾注于扩充社团规模（李谦：招新破纪录）或重塑社团文化与制度（周熠嘉：文化 ∞，建立数字架构）。**允许特定维度的 C，正是树莓社尊重历史、实事求是，推崇“在对的时间做对的事”的管理哲学的最佳体现。**
